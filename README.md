## 🏗️ Backend Architecture

> A typical request flow through my FastAPI applications.

```mermaid
graph TD
    Client["🌐 Client (HTTP Request)"]
    FastAPI["⚡ FastAPI\n(Routes + Middleware)"]
    Auth["🔐 Auth Layer\n(JWT Validation)"]
    Service["🧩 Service Layer\n(Business Logic)"]
    SQLAlchemy["🗄️ SQLAlchemy ORM\n(Query Builder)"]
    PostgreSQL["🐘 PostgreSQL\n(Database)"]
    Docker["🐳 Docker Container"]

    Client --> FastAPI
    FastAPI --> Auth
    Auth --> Service
    Service --> SQLAlchemy
    SQLAlchemy --> PostgreSQL
    Docker -.->|"wraps"| FastAPI
    Docker -.->|"wraps"| PostgreSQL
```
